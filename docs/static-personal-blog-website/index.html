Static Personal Blog Website

1 安装 Hugo

Hugo是一个静态网站生成器，可以将Markdown文件转换为HTML文件。 在正式安装Hugo之前，需要先安装Git, Go, 和 Dart Sass。

GIT：

  • 从来源构建Hugo
  • 使用Hugo模块功能
  • 将主题安装为Git子模块
  • 从本地GIT存储库中访问提交信息
  • 使用CloudCannon，CloudFlare页面，Github页面，Github页面，Gitlab页面和Netlify等服务托管您的网站。

安装链接如下：

Mac 下载

Windows 下载

Linux 下载

Go：

  • 从来源构建Hugo
  • 使用Hugo模块功能 下载链接：https://go.dev/doc/install

Dart Sass：

  • 使用SASS语言的最新功能时，需要Dart Sass将Sass转移到CSS。

    下载链接：https://gohugo.io/functions/css/sass/#dart-sass

在以上三个软件安装完成后，就可以开始安装Hugo了。推荐安装extended/deploy edition。

命令行输入：

CGO_ENABLED=1 go install -tags extended,withdeploy github.com/gohugoio/hugo@latest

2 快速开始

浏览器打开Hugo官网：https://gohugo.io/，点击主题菜单栏选择喜欢的网页模版。

3 托管到github

把我们本地生成的静态网页托管到github上进行保存，而不用自己进行维护。参考这篇。更新指令如下：

(base) ➜  newsite git:(main) ✗ hugo --minify --theme=docura --baseURL="https://Greletudiant.github.io"
(base) ➜  newsite git:(main) cd ./public
(base) ➜  public git:(main) ✗ git add .
(base) ➜  public git:(main) ✗ git commit -m "update"
(base) ➜  public git:(main) ✗ git push origin main

这是我的github仓库地址：https://github.com/Greletudiant/bstass2.git

GitHub Pages静态文件发布

1. 设置GitHub Pages

在仓库的设置页面（Settings），找到“GitHub Pages”部分。 在“Source”下拉菜单中，选择“main branch”（或“master branch”，取决于你的仓库设置）。 点击“Save”按钮。

2. 访问网站

在“Source”下拉菜单中，选择“main branch”（或“master branch”，取决于你的仓库设置）。 点击“Save”按钮。 等待一段时间，GitHub Pages会自动构建你的网站。 访问你的网站地址：URL_ADDRESS访问你的网站地址：https://yourusername.github.io。

具体操作步骤参考这篇

Netlify对托管到github上的静态网页进行渲染

1 Netlify官网： 官方首页：https://www.netlify.com/

2 首先是到netlify申请注册一个账号。这里可以使用Github、Gitlab以及Bitbucket直接授权登陆。然后登录到空间管理中心，点击右上角的"New site from Git"添加网站。

3 然后根据自己的托管平台，可以选择GitHub、GitLab或者BitBucket。我们以最常用的GitHub为演示例子。（选择GitHub的同学，别忘了勾选下方的"Limit GitHub access to public repositories.“选项）

4 点击GitHub之后会弹出一个让你授权的窗口，然后点击"Authorize netlify"之后，就会在netlify中读取你所有的代码库。

5 点击你已经建好的库，选好分支（默认master即可），然后点击"Deploy site”，系统就会自动编译你的静态页面了。同时还会给出你的页面二级域名等信息。

6 点击创建后，稍等一会儿，你就可以看到Netlify免费静态空间已经创建成功了，同时你在GitHub的代码也成功在Netlify运行了。
